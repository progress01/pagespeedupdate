# Google PSI 網站效能批次檢測
基於 Google PageSpeed Insights API 的批次檢測工具。

## 核心功能
- 批次檢測：支援一次輸入多個網址，自動依序檢測「行動版」與「電腦版」分數。
- 智慧貼上：直接從 Excel 或文件複製一整排網址，點擊按鈕自動格式化並抓取網站名稱。
- 報告匯出：一鍵下載 .xlsx Excel 報表，內含所有指標分數與官方報告連結。
- 官方連結生成：檢測結果附帶 Google 官方報告連結，方便團隊溝通討論。
- 穩定重試機制：內建 API 流量限制 (429) 自動冷卻與重試機制，減少失敗率。

## API Key 說明
為了獲得穩定的檢測體驗，強烈建議使用自己的 Google API Key。
- 安全性：您的 Key 僅會暫存在您的瀏覽器記憶體中，不會上傳到任何第三方伺服器。
- 費用警示：Google PSI API 每日有免費額度（約 25,000 次），請留意 GCP 後台配額，避免產生額外費用。
申請教學：工具介面內建圖文模擬教學，點擊「如何申請？」即可查看。
